What is BlindHelper

This is a program designed to make internet usage more convenient for visually impaired individuals. When a user specifies an area on the screen through dragging, BlindHelper reads the text within that area, describes the images, and provides a summary.

Who We Are

We are a duo of KAIST computer science students. Participating in Mad Camp 2023, our interest in improving accessibility for the blind was shared, and we started this project. This led us to the development of BlindHelper.

How to use

You should get License with BlindHelperLicense.exe
Then, you can use BlindHelper.exe
Ctrl + alt + q : Enter capture mode
Click anywhere in capture mode: Cancel capture mode
Drag in Capture Mode: Specify Zone
Ctrl + alt + w : Cancel description
Ctrl + alt + e : Pause description
Ctrl + alt + r : play the description again
Ctrl + alt + d : End of program
Ctrl + alt + f : Start summary
